Widget Configuration

I decided not just to complete the given task by making static widget to satisfy task description, but to create a configurable, reusable component. You can configure it via passing configuration object to widget's constructor:

    var widget = new CircleChart({
        // Config options go here.
    });

    widget.init();

As a result I got a widget which dynamically calculates positions and sizes of it's inner elements. But everything has it's cost. In my case I paid with an additional amount of code which do nothing but making calculations of coordinates and sizes. This part code is not really interesting to read so I hope you won't hit me too much :)

Options you can pass to your configuration object: 

    size

It's a size (diameter) of the circle. Default value is 200. The sizes and positions of other widget components are dynamically calculated according to this parameter. 

    parent

The identifier of html element that widget will be applied to. For example: 'p', 'div.myClass', '#myDiv'. Default value is 'body'. 

    thickness

The width of colorful circle borders. Default value is 10. 

    primaryColor
    secondaryColor

The default colours of chart. If no colours are found for i-th chart, these colours will be set. Default values are '#005E00' and '#4DD027' for primaryColor and secondaryColor respectively. 

    colors

Array of colours. i-th element of array represents colour pair for i-th chart. Array elements have format: {primary: '#...', secondary: '#...'}. Default value is []. 

    fontFamily

Font for all text labels inside the widget. Default value is 'sans-serif'. 

    titleSize

Sets the font size for chart's title. Default value is '20px'. 

    sumSize

Sets the font size for chart's value pair's sum text. Default value is '28px'. 

    descriptionSize

Sets the font size of text that is displayed on the bottom of the chart. Default value is '14px'. 

    view

Sets one of two widget modes. Possible values: 'simple' and 'carousel'. Default value is 'simple'. 

    data

Object which contains chart data. All calculations and widget's behaviour depends on this property. Should be provided by developer. The graph inside the chart is a visual representation of data object.

More examples? A single chart.

    var singleChart = new CircleChart({
        parent: '#example3',
        data: [ // One array element
            {
                title: 'Title',
                member1: 'First',
                member2: 'Second',
                values: [
                    {m1: 250, m2: 350},
                    {m1: 100, m2: 800},
                    {m1: 100, m2: 800}
                ]
            }
        ]
    });
    singleChart.init();

Changing size of circles

    {
        parent: '#example4',
        size: 260,
        colors: [
            {primary: '#8F1515', secondary: '#F2E529'},
            {primary: '#605DB0', secondary: '#5DB081'}
        ],
        data: data
    }

Also changing size of borders...

    {
        parent: '#example5',
        size: 260,
        thickness: 30,
        colors: [
            {primary: '#8F1515', secondary: '#F2E529'},
            {primary: '#605DB0', secondary: '#5DB081'}
        ],
        data: data
    }

... and font sizes

    {
        parent: '#example6',
        size: 260,
        thickness: 30,
        descriptionSize: '18px',
        sumSize: '20px',
        titleSize: '12px',
        colors: [
            {primary: '#8F1515', secondary: '#F2E529'},
            {primary: '#605DB0', secondary: '#5DB081'}
        ],
        data: data
    }

Let's make a carousel from it

    {

        thickness: 30,
